Características de XML

  1. Se permiten mayúsculas, independientemente de la plataforma
  2. Etiquetas no definidas, como carateres en texto plano
  3. Sin estilo, es decir es flexible y fácil de usar
  4. Las distintas partes del docuento están delimitadas por marcas
